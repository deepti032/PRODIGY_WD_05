🌤 Weather App

A simple and modern weather application that fetches real-time weather data using a weather API. Users can search for any city or use their current location to get accurate weather details.



🚀 Features

* 🔍 Search weather by city name
* 📍 Get weather using current location
* 🌡 Displays temperature in Celsius
* 🌥 Shows weather condition with icons
* 💧 Humidity and 🌬 wind speed info
* ⚡ Loading indicator and error handling
* ⌨️ Press "Enter" to search quickly
* 🎨 Clean and modern glass UI



 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
* OpenWeather API

🔑 Getting Started

 1. Clone the repository

```
git clone https://github.com/your-username/weather-app.git
```

 2. Navigate into the project folder

```
cd weather-app
```

3. Get your API key

* Go to: https://openweathermap.org/api
* Sign up and generate your free API key

4. Add your API key

Open `script.js` and replace:

```
const apiKey = "YOUR_API_KEY_HERE";
```



 ▶️ Run the App

Simply open `index.html` in your browser.
